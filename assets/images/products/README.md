# Imágenes de productos (`assets/images/products/`)

Copias **optimizadas para web** (recortadas/comprimidas) de las fotos oficiales,
para usar como referencia o thumbnails del configurador.

- **Originales** (galería oficial): `_entrada-local/referencias-productos/<producto>/`.
- **Descarga:** `bash _entrada-local/referencias-productos/descargar-referencias.sh`.
- El entorno de generación **no permite descargas binarias por red**, por eso esta
  carpeta queda lista pero **vacía**: las URLs reales están en cada `producto.json`.
